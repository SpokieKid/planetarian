With Aaros: Planetary Pet 星球养成游戏

🌌 项目目标

打造一个完整的前端项目，支持用户养成属于自己的星球。星球出生于不同社会模式（International / Global / Planetary），并在成长过程中经历不同资源变化与环境演化，同时融入我们强调 "Planetary Onchain Society" 的叙事内容。

🔧 技术要求

  • 【前端框架】React + Zustand 状态管理
  • 【渲染】Phaser.js 或 Pixi.js（用于星球与环境变化渲染）
  • 【交互】轻量 Canvas / WebGL 动效，支持点击、拖拽
  • 【数据管理】本地持久化（LocalStorage），预留接口用于未来链上存档
  • 【部署】网页端（初期），未来可兼容 Farcaster Frame / Telegram MiniApp

🌍 核心功能模块

1. 星球生成（Planet Creation）

  • 用户初次进入时选择出生模式：
    • International
    • Global
    • Planetary
  • 每种模式初始星球外观、资源设定不同。
  • 自动生成星球名称（可随机或用户自定义）。

2. 星球养成系统（Planet Evolution）

  • 星球每日/每次交互积累成长点（Growth Points）。
  • 不同模式影响成长逻辑：
    • International：受限多，成长慢，需频繁审批、验证（模拟国家壁垒）。
    • Global：中等成长速度，存在费用/资源消耗（模拟全球资本流动）。
    • Planetary：无阻碍，自由成长，资源繁荣。

3. 资源系统（Resources）

  • 资源包括：
    • 水（Water）
    • 光（Light）
    • 气（Air）
    • 思想（Ideas）
  • 每日资源获取受社会模式影响：
    • International：资源受限（如有配额或关税）
    • Global：资源浮动（如需手续费）
    • Planetary：资源自由增长（链上验证后可额外奖励）

4. 环境变化（Environment Shifts）

  • 随着成长，不同模式下星球环境发生变化：
    • International：星球有国界线、审查云、孤岛城市。
    • Global：跨国枢纽、金融风暴、巨型贸易中心。
    • Planetary：自由丛林、链上城市、无边际光带。
  • 可通过不同 UI/动画展示。

5. 叙事内容嵌入（Narrative Integration）

  • 在特定成长节点触发叙事事件（如："你的星球因开放治理而迎来繁荣"）。
  • 叙事文本短小精炼，强调"去中心化自由"、"Planetary 未来"等主题。

6. 交互与小游戏（Mini-Interactions）

  • 简单交互增加粘性：
    • 给星球添加能量（水、光、思想等）
    • 升级星球文明（需消耗资源）
    • 选择星球发展的方向（如：文化优先、科技优先）

7. 分享与社交（Sharing）

  • 用户可生成当前星球快照，分享到 Farcaster 或 Telegram
  • 未来支持 mint 星球为 NFT（可预留接口）

🌟 开发优先级建议（阶段划分）

Phase 1（黑客松目标）

  • 完成星球模式选择 + 初始生成
  • 完成资源变化与简单环境演化（每种模式至少两阶段变化）
  • 基础交互：添加资源、查看状态
  • 简单叙事文本触发（如 Planetary 模式成长快的提示）

Phase 2（后续迭代）

  • 更丰富的环境演变动画与艺术素材
  • 更多资源种类与事件（如星际迁徙、文化兴起）
  • AI助理（链上治理官、文化引导者）
  • 链上身份绑定（星球钱包）与资产化（NFT化）

关于星球的动效：

✨ 星球成长动效设计提案

总体要求

  • 风格：简洁轻量，突出成长感（避免重型 3D 模型负担）。
  • 引擎：Phaser.js 或 Pixi.js 渲染。
  • 动画风格：缓慢律动（类似 Mountain 风格），增加沉浸感。
  • 动效触发：成长节点变化、资源添加、模式转变。

🪐 动效元素拆分

动效类型描述技术细节
星球基础旋转星球缓慢自转（周期 20s-30s）sprite.rotation += 0.001 (Phaser 示例)
资源灌溉动画添加水、光、思想等资源时，星球表面出现光晕或粒子流动使用 ParticleEmitter 或 Alpha 渐变过渡
环境阶段演变随着成长，星球表面解锁新的地貌元素（如森林、城市、链上节点）分层 Sprite 替换或动态叠加
模式切换动效切换 International ➔ Global ➔ Planetary 时，星球颜色、背景星空发生变化场景渐变 + 星球外形变化
叙事事件触发某些叙事节点出现时，短暂高亮动画提示加强粒子效果/屏幕轻微闪烁
星球健康反馈若资源匮乏，星球表面出现裂痕、乌云；资源充足时星球表面明亮生机贴图替换+渐变动画

📋 具体动效流程示例

例：玩家添加“思想资源”

  • 点击思想按钮 ➔
  • 星球表面出现淡紫色粒子流（轻微上升扩散）➔
  • 星球整体 Alpha 值微微增强 ➔
  • 思想条上涨 ➔
  • 弹出轻提示：“你的星球思想之光正在闪耀 🌟”
